# Cubic Sandbox script using Game Guardian
This repository is dedicated to my old Lua script for the online game Cubic Sandbox. I wrote it when I was 13. Back then, a developer named "dem" was popular in our community. He wrote cool scripts for that game and posted videos on YouTube. I decided to give it a try. The main challenge was searching for values ​​in the game through Game Guardian using offsets. For example, nickname length was limited to 20 characters. First, you enter a nickname of the maximum length, search for a value of 20, then change the length and search for a new value. After a couple of tries, you'd find the exact value that determined the nickname length and then change it. Sometimes, searching for values ​​was very difficult. Since there were multiple regions with values, you had to guess which region the value was in. If there were too many values, Game Guardian would simply crash, making the search even more difficult. But despite everything, I was interested. Then I wrote my first version of the script and posted it on YouTube. After that, there were four more versions where I expanded the script and made it more functional. It got to the point where dem himself started exchanging the values ​​he'd found with me. I became more popular in the community. Although I wrote one of the most functional scripts among the others, I still envied dem — he was even younger than me but already had so much skill. The next stage was to write a real cheat with a graphical interface using Java. But alas, I still couldn't figure out how to do that; I didn't even know what offsets were back then. In fact, the Cubic Sandbox community was supported solely by us and a few other people who were also popular. And even though we've all left the game and many years have passed, it's still running, and new cheaters occasionally pop in and terrorize players. The game has simply become boring without all those people. But I don't regret the effort; I've met many interesting people and like-minded people. I still remember the first time I logged into that game. Anyway, this repository is just there to show anyone who's interested how cheats worked on Game Guardian. It's more of a historical repository that won't be updated.
